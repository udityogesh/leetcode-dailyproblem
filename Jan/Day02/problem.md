## Problem Summary
You are given an integer array `nums` of size `2N`.
Exactly **one element** in the array is repeated **N times**, while all other elements appear **only once**.

## Task
Find and return the element that is repeated `N` times.

## Rules
- The array size is always even (`2N`)
- Only one element is repeated `N` times
- Order of elements does not matter

## Example
Input: `[1,2,3,3]`  
Output: `3`

Input: `[2,1,2,5,3,2]`  
Output: `2`

## Key Point
There is **guaranteed** to be exactly one valid answer.
